CASE 1: Project delete ho gaya laptop se → GitHub se wapas lao

Agar GitHub par already upload hai, tension nahi.

Step-by-step recover project
1️⃣ GitHub repo link copy karo

Example:

https://github.com/kkhansameer94/CodeCrush.git

2️⃣ Desktop ya jahan project chahiye wahan open terminal (CMD)
cd Desktop

3️⃣ Clone command
git clone https://github.com/kkhansameer94/CodeCrush.git

4️⃣ Folder open karo
cd CodeCrush


Bas 🎉
Poora project wapas aa jayega same as GitHub.

cd Desktop
